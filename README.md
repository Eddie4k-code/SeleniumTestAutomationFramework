# pyTestAutomationFramework
A Data Driven Java-based Selenium Automation Testing Framework for a demo banking application.
https://www.globalsqa.com/angularJs-protractor/BankingProject/#/customer

<h1>Description</h1>
There are two tests added to this framework,
1.Login Test - Tests logging in as a regular user and manager.
<br></br>
2.AddCustomer Test - Adds a customer to the banking application

<h1>Directory</h1>
<h3>pageObjects</h3> - Contains all pages that will be tested on the web application includes their elements and relevent methods.
<h3>utils</h3> -  Contains Constants, Excel Utility, FrameworkProperties and Reporting Listener.
<h3>drivers</h3> - Contains web driver (in this case only chromedriver)
<h3>testScripts</h3> - Contains all test scripts for the banking application.
<h3>testData</h3> - Contains csv files that are used for test data when a test is active.
<h3>Reports</h3> - Contains all html generated reports
<h3>Screenshots</h3> When a test fails a screenshot will be saved here showing the last step it was on.
<h3>Log</h3> Contains logs from test runtime.
